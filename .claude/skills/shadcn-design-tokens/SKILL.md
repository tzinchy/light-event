# Дизайн-токены и визуальный язык

Основа компонентов — shadcn/ui + Tailwind, токены темы приведены к палитре референса. Mobile-first (кабинет соискателя — по сути мобильный экран, 390px).

## Палитра (нейтральный zinc + зелёный акцент)

- Фон/текст: `#18181b` (инверсия — `#fafafa`); поверхности `#fafafa` / `#f4f4f5`; границы `#e4e4e7`; вторичный текст `#52525b` / `#a1a1aa`.
- Акцент: зелёный `#16a34a` / `#15803d`, фон `#f0fdf4`, граница `#bbf7d0`.
- Статусы: успех — зелёный; ошибка/ЧС — красный `#b91c1c` / `#fef2f2` / `#fecaca`; предупреждение (review/draft/pending) — янтарный `#b45309` / `#fffbeb` / `#fde68a`; инфо (paid/processing) — синий `#1d4ed8` / `#eff6ff` / `#bfdbfe`; бейдж «Тест компании» — фиолетовый `#7c3aed` / `#f5f3ff` / `#ddd6fe`.

## Типографика и форма

- Шрифт Geist; Geist Mono — для сумм, OTP-ячеек, кодов ссылок.
- Крупные скругления (8–14px), чистый минимализм, карточки.
- Паттерны референса: статус-бейджи (11px, 600, цветной фон+граница), сегмент-контролы (список/карта), чипсы-фильтры (активный — инверсия `#18181b`), степперы, прогресс-бары заполняемости, нижний таб-бар у соискателя, сайдбар у организации/админа.
- Локаль — русский; тексты брать из словаря референса (DICT в PLAN.md/референсе).
- Пустые состояния — честные (см. skill real-data-only).
