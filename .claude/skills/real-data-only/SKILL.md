# Только реальные данные

- Запрещено генерировать фейковые/seed/mock записи в рантайме и в dev/prod БД.
- Нет данных → UI показывает пустое состояние («Пока нет данных», «Пока нет заявок», «Нет диалогов»), API возвращает пустой список `[]` — а НЕ выдуманные значения.
- Единственное исключение — фикстуры внутри тестов (testcontainers), которые не попадают в dev/prod БД.
- HTML-референс содержит демо-данные — это источник структуры экранов, полей, статусов и текстов, но его строки НЕ копируются в базу.
- Счётчики/агрегаты (рейтинг, кол-во смен, статистика дашбордов) считаются из реальных записей; при их отсутствии — 0/«—», не выдуманные числа.
- Внешние провайдеры (SMS, OAuth, платежи) мокаются за интерфейсом, но не подделывают данные пользователя.
